# Sistema Bancário em Python

Sistema bancário simples via terminal feito em Python para praticar lógica de programação e manipulação de funções.

## Funcionalidades

- Depositar dinheiro
- Sacar dinheiro (com validação de saldo)
- Consultar saldo
- Ver histórico de transações
- Menu interativo no terminal

## Tecnologias

- Python 3

## Como executar

1. Clone o repositório
```bash
git clone https://github.com/lfernandes-dev/sistema-bancario-python.git
```

2. Acesse a pasta
```bash
cd sistema-bancario-python
```

3. Execute
```bash
python banco.py
```

## Exemplo de uso

```
1 - Depositar
2 - Sacar
3 - Ver saldo
4 - Ver histórico
5 - Sair
Digite um dos números acima: 1
Digite o valor que deseja depositar: 500
Depósito efetuado com sucesso! Seu saldo agora é de R$500.0
```
